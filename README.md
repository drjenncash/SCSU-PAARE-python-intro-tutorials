## Primary Purpose:

These tutorials are designed to introduce our project participants to the Python programming language, emphasizing Data Science with a slant toward the Astronomy of variable sources. 

It is expected that the notebooks will also be valuable as a starting place for others learning Python for astronomy. 

The content inside these tutorials draws heavily from a wide variety of websites, user guides, and existing tutorials.

## Detailed Curriculum Outline

### Module 1: Foundations

**Module Goal:**
Module 1 of the tutorials will build the skills needed to do basic python work inside a jupyter notebook. In general this module will be similar to most introductory python tutorials with the exception that the examples and exercises lean toward the skills most commonly used in Astronomical Data Analysis. 

Users with no previous programming experience will need to work through the tutorials carefully. If they are working alongside of more experienced users, they should ask questions for clarification as needed. AI such as ChatGPT or the AI assistant available in the Anaconda in the Cloud platform can be used, but beginning users should specifically ask the AI to explain each step of the procedure. 

Users with previous programming in another language should progress through these tutorials fairly quickly and should focus on some of the python syntax for programming concepts.

Users with previous python experience may be able to skip over the tutorials in this module. They are encouraged to try the exercises at the end of each tutorial to reinforce their previous understanding.
